# Perfume

A simple Python app that helps decide which perfume to wear based on weather and gender (male/female).

## Features
- Supports **male** and **female** recommendations.
- Uses temperature and rain condition.
- Automatically maps weather to categories: `hot`, `mild`, `cold`, `rainy`.
- Shows 2 perfume suggestions with note profile and reason.

## Run
```bash
python3 perfume_app.py
```

## Test
```bash
python3 -m pytest -q
```
